getProjection

  • getProjection(): JProjection
    • Defined in node_modules/jmap-api-ng/public/api.d.ts:1752

    JMap.Service.Project.getProjection

    Returns JMap project projection.

    If no project is loaded, returns an empty projection : { code: "", name: "" }.

    In MapBox, projection is always "EPSG:3857", but that function returns the project defined projection (so it can be different than ESPG:3857).

    example
    
// returns the project projection
JMap.Service.Project.getProjection()

    Returns JProjection

load

  • load(projectId?: undefined | number): Promise<void>
    • Defined in node_modules/jmap-api-ng/public/api.d.ts:1866

    JMap.Service.Project.load

    You can display another project by calling this method.

    User session rigths are checked server side and an error is thrown if user doesn't have the access right for the project.

    If no projectId is provided, the one defined in API options (JAPIOptions) will be used. If no project id has been defined, an error is thrown.

    throws

    Error if project not found

    example
    
// Load data and display map of project id=2
JMap.Service.Project.load(2)

    Parameters

    • Optional projectId: undefined | number

      The JMap project id

    Returns Promise<void>

    a promise that is resolved when the project has been loaded successfully
